  • Let's Go

    New to Dubai? Looking for places you haven't visisted yet? If yes,then check out Let's Go! It's an android application that makes use of the ARCGIS SDK & OSM to help the user get around Dubai by providing directions with different transport modes (Walking,Public Transit and by Car). It also features a scheduling section where the user can "pin" a location they want to visit and strike it out once they've visited.
